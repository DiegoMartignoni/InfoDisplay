Tecnologie e linguaggi
======================

Linguaggi
---------
- PHP
- SQL
- HTML
- CSS
- JavaScript

La piattaforma (provvisoria) utilizzata per il servizio di hosting è **Altervista**. Il framework utilizzato per una migliore gestione è  **CodeIgniter**.

Prerequisiti
------------
I requisiti minimi sono gli stessi richiesti dal framewok `CodeIgniter <https://www.codeigniter.com/user_guide/general/requirements.html>`_.
